倒立振子 cart-pole — 上向きの不安定平衡を立てる MPC

台車を1本の力 F で動かし、その上のフリーな振子を倒立させ続ける。本質的に不安定なプラントを、線形化モデルで未来を予測しながら安定化。力には上限（飽和）。横から蹴られても立て直す。

モデル化

台車＋振子の力学 → 線形化 → 離散化
台車を押すと上の振子が倒れようとする不安定な連成系。状態は台車の位置 x・速度 ẋと振子の角度 θ・角速度 θ̇の4つ。入力は台車を押す力 F 1つだけで、倒立維持と台車の定位置保持を両立させたい。
運動方程式（台車＋振子・非線形 → θ≈0 で線形化）
\(\ddot{x} \approx \dfrac{1}{M+m}\,F - \dfrac{m g}{M+m}\,\theta\)＝つまり台車は力Fで動き傾き θ に少し引かれる
\(\ddot{\theta} \approx \dfrac{(M+m)g}{L\left(\tfrac{4}{3}(M+m)-m\right)}\,\theta - \dfrac{1}{\,\cdots\,}\,F\)＝つまり倒れるほどさらに倒れる（正係数＝不安定）。押すと逆へ傾く
→ 状態空間 ẋ = A x + B F　（x = [x, ẋ, θ, θ̇]ᵀ）
\[\begin{bmatrix}\dot{x}\\\ddot{x}\\\dot{\theta}\\\ddot{\theta}\end{bmatrix}=\begin{bmatrix}0&1&0&0\\0&0&\textcolor{#b9760f}{-1.40}&0\\0&0&0&1\\0&0&\textcolor{#b9760f}{16.82}&0\end{bmatrix}\begin{bmatrix}x\\\dot{x}\\\theta\\\dot{\theta}\end{bmatrix}+\begin{bmatrix}0\\\textcolor{#2F7DD1}{0.952}\\0\\\textcolor{#2F7DD1}{-1.429}\end{bmatrix}F\]
左の縦ベクトル＝各状態の変化。2行目 ẍ・4行目 θ̈ が上の式そのもの。橙＝傾き θ の効き（+16.82＝倒れる側＝不安定）、青＝力 F の効き。実装は数値微分で A,B を算出。
離散化（ZOH・dt = 0.05 s）
\(x_{k+1} = A_d x_k + B_d F_k\)\(A_d=e^{A\,dt}\)（数値ヤコビアンで A,B 構築）
M = 1.0 kgm = 0.2 kgL = 0.5 mg = 9.81dt = 0.05 s状態 4次

制御ロジック（MPC）

4状態を罰する予測最適化
MPCは角度 θ を最も重く罰するコストで、未来 Np=40 ステップ（≒2秒）を予測し、力の加え方を整形。倒れる前に先回りして立て直し、同時に台車も中央へ戻す。線形なので事前計算ゲインで実行。
毎ステップ解く最適化（＋制約 |F|≤Fmax）
\[\min_{F}\ \sum_{k=1}^{N_p}\left(Q_\theta\,\theta_k^2 + Q_x (x_k-x^*)^2 + R\,F_k^2\right)\quad \text{s.t. } |F|\le 15\,\text{N}\]
Q = diag(8, 1, 60, 6) → 角度 θ を最も重く罰する（R=0.02）
比較対象：PID（PD）制御則
\[F_{\text{PID}} = 70\,\theta + 14\,\dot{\theta} + 0.9\,(x-x^*) + 1.3\,\dot{x}\]
角度と位置の比例・微分のみ（固定ゲインのPD）。未来予測も制約も持たない（③で挙動を比較）。
  • 4状態すべてを重み付き評価し、姿勢と台車位置を同時に整える
  • 未来 Np=40（≒2秒）を予測・Nc=12 で力の列を最適化し先頭だけ実行（線形なので事前計算ゲイン \(F = K_r x^* - K x\)）
  • PIDとの違い：PID は今の角度・位置に反応するだけ。MPCは未来を予測し制約内で先回りする（Np 過小は不安定化）
  • 状態の入手：MPCは4状態 x が必要。角速度などが測れない実機ではカルマンフィルタ／オブザーバで推定して −K に渡す

制御の効果

外乱キック：MPC vs PID
読み方：t=1.5s で振子に横蹴り（赤帯）を入れたとき、姿勢の立て直しと台車の戻りを MPC と PID（同じ力上限）で比較——MPC は素早く整定し台車もほぼ中央、PID は整定が遅く台車が大きく流れる。

▶ 上のデモで試す
  1. 外乱（蹴る）で倒立を崩し、制約内で立て直す動きを見る
  2. 予測ホライズン Np を下限まで下げる——②の「Np 過小は不安定化」を体感
  3. 力の上限 Fmax を下げる——同じ外乱でも復帰がぎりぎりになる
