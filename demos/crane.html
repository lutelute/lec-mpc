仕組み

状態 x=[位置,速度,φ,φ˙]（φ＝吊り角）。 MPC のコストは位置誤差だけでなく振れ角 φ も罰するので、加速と減速のタイミングを予測し、目標で振れが消えるよう運ぶ（最適な入力整形）。 OFF にすると位置だけ見る PD になり、積み荷が大きく揺れます。

モデル化

台車＋振子の連成系を1本の力で
台車を動かすと吊り荷が振り子のように揺れる連成系。状態は台車の位置 x・速度 ẋと振れの角度 φ・角速度 φ̇の4つ。入力は台車を押す力 F 1つだけで、位置合わせと振れ消しを両立させたい。
運動方程式（微小振れ sinφ≈φ, cosφ≈1 で線形化）
\(\ddot{x} \approx \dfrac{1}{M_t}F + \dfrac{m_p g}{M_t}\varphi\)台車は力Fで動き＋振れφに引かれる
\(\ddot{\varphi} \approx -\dfrac{(M_t+m_p)g}{L\,M_t}\varphi - \dfrac{1}{L\,M_t}F\)振れは戻ろうとし押すと逆へ
→ 行列形式 ẋ = A x + B F　（x = [x, ẋ, φ, φ̇]ᵀ）
\[\begin{bmatrix}\dot{x}\\\ddot{x}\\\dot{\varphi}\\\ddot{\varphi}\end{bmatrix}=\begin{bmatrix}0&1&0&0\\0&0&\textcolor{#b9760f}{2.94}&0\\0&0&0&1\\0&0&\textcolor{#b9760f}{-21.3}&0\end{bmatrix}\begin{bmatrix}x\\\dot{x}\\\varphi\\\dot{\varphi}\end{bmatrix}+\begin{bmatrix}0\\\textcolor{#2F7DD1}{1}\\0\\\textcolor{#2F7DD1}{-1.67}\end{bmatrix}F\]
左の縦ベクトル＝各状態の変化。2行目 ẍ・4行目 φ̈ が上の式そのもの。橙＝振れ φ の効き（−21.3＝復元）、青＝力 F の効き
離散化（ZOH・dt = 0.05 s）
\(x_{k+1} = A_d x_k + B_d F_k\)\(A_d=e^{A\,dt}\)（数値ヤコビアンで A,B 構築）
L = 0.6 mMₜ = 1.0mₚ = 0.3g = 9.81dt = 0.05 s状態 4次

制御ロジック（MPC）

振れ角 φ を罰する予測最適化
MPCは振れ角 φ を重く罰するコストで、未来 Np=40 ステップ（≒2秒）を予測し、力の加え方を整形。到着時に振れが消えるように先読みして加減速する（アンチスウェイ）。
毎ステップ解く最適化（＋制約 |F|≤Fmax）
\[\min_{F}\ \sum_{k=1}^{N_p}\Big((x_k-r)^\top Q\,(x_k-r) + r\,F_k^2\Big)\quad \text{s.t. } |F|\le 8\,\text{N}\]
Q = diag(10, 1, 40, 4) → 振れ角 φ を最も重く罰する
比較対象：PD 制御則（位置のみ）
\[F_{\text{PD}} = 3.0\,(x_{\text{ref}}-x) - 2.0\,\dot{x}\]
位置誤差の比例と速度の微分だけ。位置だけ見て振れ角 φ を考えない＝揺れを置き去り（③で挙動を比較）。
  • 未来 Np=40 ステップ（≒2秒）を予測、Nc=12 で力の列を最適化し先頭だけ実行
  • 到着時に振れが消えるよう加減速のタイミングを整形（1入力で2目的）
  • PDとの違い：PD は位置だけ追い振れは置き去り。MPCは φ も予測して能動的に消す
  • 状態の入手：MPCは状態 x が必要。出力しか測れない実機ではカルマンフィルタ／オブザーバで推定して −K に渡す
