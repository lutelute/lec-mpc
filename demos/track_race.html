ねらい

MPC は壁（位置の上限）と力の上限を最適化の制約として知っているので、壁の手前で滑らかに停止。 PID は制約を知らず、目標へ突き進んで壁を越える（制約違反）。これが制約を陽に扱える MPC の最大の利点です。理論は §5 制約と QP。

台車レース — 壁を守って目標へ走る MPC

台車を力で押して目標位置へ運ぶ。力には上限があり、前方には越えてはならない壁（位置の上限）がある。MPC は壁を制約として知り、手前で滑らかに止まる。

モデル化

台車の運動 → 状態空間 → 離散化
台車を力 F で押して目標位置 r へ運ぶ。状態は台車の位置 x・速度 vの2つ。摩擦なしの2重積分系（力→加速度）で、前方には越えてはならない壁（位置の上限）がある。
運動方程式（ニュートンの法則・連続時間）
\(m\ddot{x} = F\)摩擦なし。質量 m=1 と正規化 → 入力 u=F（加速度指令）、ẍ = u/m
\(\dot{x} = v,\ \ \dot{v} = u\)位置 x と速度 v の 2 重積分系（力→加速度）
→ 行列形式 ẋ = A x + B u，y = C x　（x = [x, v]ᵀ）
\[\begin{bmatrix}\dot{x}\\\dot{v}\end{bmatrix}=\begin{bmatrix}0&\textcolor{#0C447C}{1}\\0&0\end{bmatrix}\begin{bmatrix}x\\v\end{bmatrix}+\begin{bmatrix}0\\\textcolor{#2F7DD1}{1}\end{bmatrix}u\]
1行目 ẋ=v、2行目 v̇=u がそのまま。紺＝速度が位置を動かす、青＝入力 u（力）が速度を動かす。出力 y=x（台車の位置）、C=[1, 0]。
離散化（ZOH・dt = 0.1 s）— 2重積分は厳密
\(A_d=\begin{bmatrix}1&dt\\0&1\end{bmatrix}\quad B_d=\begin{bmatrix}\tfrac{dt^2}{2}\\dt\end{bmatrix}\)
\(x_{k+1}=A_d x_k+B_d u_k\)　Ad=eA·dt, Bd=∫₀dteAsds·B（2重積分は打切りなしで厳密）

制御ロジック（MPC）

予測＋制約付き QP
MPC は未来を予測し、力の上限を守りつつ壁を越えない位置列を毎瞬間計算して先頭だけ実行する。壁を制約として知るので、手前で早めに減力して滑らかに止まる。
毎ステップ解く最適化問題（追従＋省力）
\[\min_{U}\ \sum_{i=1}^{N_p}\left(r - y_{k+i}\right)^2 + \lambda\sum_{j=0}^{N_c-1} u_{k+j}^2\]
＝目標 r と予測位置 y の誤差を小さく、かつ力 u を穏やかに（重み λ）。
制約（守るべき上限）— QP の不等式
\(-F_{\max}\le u_{k+j}\le F_{\max}\)力の上限（飽和を最適化に内蔵）
\(y_{k+i}\le x_{\text{wall}}\)位置の上限＝壁。予測位置が壁を越えない
比較対象：PD 制御則（壁を知らない）
\[u_{\text{PD}} = K_p\,(r - x) - K_d\,v\qquad(K_p=1.5,\ K_d=2.5)\]
位置誤差 r−x の比例と速度 v の微分制動のみ。壁 x≤xwall を制約として扱えず、目標へ突進して壁を越える（力は ±Fmax でクランプするだけ）。
  • 未来 Np=20 ステップを予測、制御 Nc=6 手を最適化し先頭だけ実行
  • QP は Hildreth 法で求解（重み λ=0.1、毎ステップ反復）
  • PDとの違い：PD は壁を知らず突進して越える。MPC は壁を制約として持ち手前で減力
  • 状態の入手：MPCは状態 x が必要。出力しか測れない実機ではカルマンフィルタ／オブザーバで推定して −K に渡す

結論：MPC は壁を制約として知り、手前で早めに減力して滑らかに停止する。PID は壁を知らず目標へ突進し、壁を越えて制約違反になる。これが制約を陽に扱える MPC の最大の利点。
