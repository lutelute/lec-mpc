仕組み

状態 [egap,Δv,aego]（車間誤差・相対速度・自車加速度）。 MPC は車間誤差を 0 にしつつ、加減速の上限を制約として守る。前車の急減速時も先読みで早めに減速し、急ブレーキ＝快適性/安全の限界内に収める。 PID 的な単純追従より追突しにくく・滑らか。

モデル化

縦運動の物理 → 状態空間 → 離散化
自車が前車との車間を目標値 d* に保つ問題。状態は車間の誤差・相対速度・自車の加速度の3つで表す。前車の加減速は外乱として効く。
運動方程式（誤差座標・連続時間）
\(\dot{e}_{\text{gap}} = \Delta v\)車間誤差の変化 ＝ 相対速度
\(\dot{\Delta v} = a_l - a_e\)前車加速度 \(a_l\) は外乱
\(\dot{a}_e = (u - a_e)/\tau\)加速度の一次遅れ
状態空間　ẋ = A x + B u + E al
\[\begin{bmatrix}\dot{e}_{\text{gap}}\\\dot{\Delta v}\\\dot{a}_e\end{bmatrix}=\begin{bmatrix}0&1&0\\0&0&\textcolor{#b9760f}{-1}\\0&0&\textcolor{#b9760f}{-1/\tau}\end{bmatrix}\begin{bmatrix}e_{\text{gap}}\\\Delta v\\a_e\end{bmatrix}+\begin{bmatrix}0\\0\\\textcolor{#2F7DD1}{1/\tau}\end{bmatrix}u\]
橙 ＝ 自車加速度の効き、青 ＝ 指令 \(u\) の効き。外乱 \(E=[0,1,0]^\top\)

\(x_{k+1} = A_d x_k + B_d u_k\)\(A_d=e^{A\,dt}\)

制御ロジック（MPC）

予測＋制約付き最適化
MPC は未来を予測し、加減速の上限を守りながら車間誤差を最小にする加速度列を毎瞬間計算し、先頭だけ実行する。前車の急減速も先読みできる。
毎ステップ解く最適化（＋制約）
\[\min_{u}\ \sum_{k=1}^{N_p}\left(q\,e_{\text{gap}}^2 + r\,u^2\right)\quad \text{s.t. } a_{\min}\le u\le a_{\max}\]
＝車間誤差 egap を小さく、かつ加減速 u を穏やかに。上限は制約として内蔵。
比較対象：PID 制御則
\[u_{\text{PID}} = K_p\,e + K_d\,\dot{e} + K_i\!\!\int\! e\,dt\]
誤差 e の比例・微分・積分のみ。未来予測も制約も持たない（③で挙動を比較）。
  • 未来 Np=25 ステップを予測して最適加速度列を計算、先頭だけ実行
  • 前車の急減速を先読みし、上限内で早めに減速
  • PIDとの違い：PIDは今の誤差にしか反応せず制約も知らない。MPCは未来と上限を同時に考える
  • 状態の入手：MPCは状態 x が必要。出力しか測れない実機ではカルマンフィルタ／オブザーバで推定して −K に渡す
